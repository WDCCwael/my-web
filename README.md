# WDC - Where Development Meets Creativity

Professional website for WDC digital marketing and web design agency.

## Tech Stack

- **React 18** - UI library
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Animations
- **Lucide React** - Icons
- **React Router** - Routing

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Pages

- **Home** - Hero, Stats, Services, Portfolio Preview, Testimonials, Contact
- **About** - Company story, core values, why choose us
- **Services** - Detailed service offerings
- **Portfolio** - Project showcase gallery
- **Blog** - Articles and insights
- **Contact** - Contact information and inquiry
